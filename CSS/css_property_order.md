# CSS 속성의 권장 작성 순서

1. 레이아웃

   1. `display`
   2. `visibility`
   3. `overflow`
   4. `float`, `clear`
   5. `position`

2. 박스모델

   1. `width`, `height`
   2. `padding`
   3. `margin`
   4. `border`

3. 배경

   - `background`
   - `font`

4. 내용

   - `content`

📌 그 외 속성은 비슷한 성격의 속성이나 상호작용 중인 속성 주변에!
