# 선택자(Selector)

## 1. 기본 선택자

### 1.1. 유형 선택자(Type Selector)

태그에 대한 스타일 지정 방법으로, 해당 태그명을 가진 모든 요소에 적용된다.

CSS 파일 상단부에 작성하는 것이 권장된다.

### 1.2. 아이디 선택자(ID Selector)

HTML 전역 속성인 `id`를 이용한 스타일 지정 방법이다. 해당 ID를 가진 모든 요소에 적용된다. 그러나 ID는 고유 식별자로 사용하는 것이 권장된다.

CSS에서 id는 `#`으로 표현한다.

### 1.3. 클래스 선택자(Class Selector)

HTML 전역 속성인 `class`를 이용한 스타일 지정 방법이다. 해당 클래스를 가진 모든 요소에 적용된다.

CSS에서 클래스는 `.`으로 표현한다.

### 1.4. 속성 선택자(Attribute Selector)

HTML 요소의 특정 속성값을 선택하여 스타일을 적용하는 방법이다.

#### 1.4.1. `[attr]`, `[attr = value]`

-
- `[attr]`: 해당 속성을 가진 모든 요소를 선택한다.
- `[attr="value"]`: 해당 속성값을 정확히 가지는 요소를 선택한다.

```css
[attr] {
  /* style rules */
}

[attr="value"] {
  /* style rules */
}
```

#### 1.4.2. `[attr^]`, `[attr$]`, `[attr*]`

- `[attr^="value"]`: 해당 value로 시작하는 속성값을 가지는 요소를 선택한다.
- `[attr$="value"]`: 해당 value로 끝나는 속성값을 가지는 요소를 선택한다.
- `[attr*="value"]`: 해당 value를 포함하는 속성값을 가지는 요소를 선택한다.

```css
[attr^="value"] {
  /* style rules */
}

[attr$="value"] {
  /* style rules */
}

[attr*="value"] {
  /* style rules */
}
```

## 2. 가상 선택자(Pseudo Selector)

실제로 존재하지 않은 요소를 선택하는 역할을 한다. 이러한 요소는 브라우저가 렌더링하는 과정에서만 생성되며, 실제 HTML 문서에 존재하지 않는다.

### 2.1. 가상 클래스 선택자

'요소의 상태'에 스타일을 적용한다.

`:`를 이용해 아래 조건을 붙여서 특정 조건의 요소에 적용한다.

#### 2.1.1. `:first-child`, `:last-child`, `:nth-child()`

- `first-child`: 부모 요소의 첫 번째 자식 요소에 적용한다.
- `last-child`: 부모 요소의 마지막 자식 요소에 적용한다.
- `nth-child()`: 부모 요소의 특정 순번의 자식 요소에 적용한다. 자연수를 넣을 수도 있고, 홀수는 '2n-1', 'odd'등, 짝수는 '2n', 'even'등을 입력할 수 있다.

> **📌주의사항**
>
> 만약 `first-child`를 사용할 때 선택된 요소가 부모의 첫 번째 자식이 아니라면 스타일이 적용되지 않는다.  
> 아래 `first-of-type`, `last-of-type`, `nth-of-type`을 사용하면 위 문제를 해결할 수 있다. 에러를 줄이기 위해서 더 권장되는 방식이다.
>
> ```html
> <section>
>   <div class="movie">Toy Story</div>
>   <p class="movie">Zootopia</p>
>   <p class="movie">Inside Out</p>
>   <div class="movie">Coco</div>
>   <span class="movie">Finding Nemo</span>
> </section>
> ```
>
> ```css
> p: first-child {
>   color: red;
> }
> /* p태그는 section의 첫번째 자식을 포함하고 있지 않기 때문에 스타일이 적용되지 않는다. */
> ```

#### 2.1.2. `:first-of-type`, `:last-of-type`, `:nth-of-type`

- `first-of-type`: 첫 번째 타입에 적용한다. 첫 번째 타입이 여러 개라면 모두 적용된다.
- `last-of-type`: 마지막 타입에 적용한다. 마지막 타입이 여러 개라면 모두 적용된다.
- `nth-of-type()`: 특정 타입에 적용한다. 자연수를 넣을 수도 있고, 홀수는 '2n-1', 'odd'등, 짝수는 '2n', 'even'등을 입력할 수 있다.

> **📌주의사항**
>
> 형제 태그 중 여러 태그가 섞여있을 경우, `first-of-type`을 사용한다면 각각의 태그 모두의 첫번째 요소에 스타일이 적용될 수 있다.  
> 위 `first-child`, `last-child`, `nth-child()`을 사용하면 위 문제를 해결할 수 있다.
>
> ```html
> <section>
>   <div class="movie">Toy Story</div>
>   <p class="movie">Zootopia</p>
>   <p class="movie">Inside Out</p>
>   <div class="movie">Coco</div>
>   <span class="movie">Finding Nemo</span>
> </section>
> ```
>
> ```css
> .movie:first-of-type {
>   color: blue;
> }
> /* Toy Story, Zootopia, Finding Nemo 모두 스타일이 적용되어 버린다. */
> ```

#### 2.1.3. `:not()`

특정 요소를 제외하고 스타일을 적용한다. 예를 들어, 첫번째 자식 요소가 아닌 `<div>`를 선택하고 싶다면 다음과 같이 표현한다.

```css
div:not(:first-child) {
  /* style rules */
}
```

속성을 기준으로 제외하고 싶을 때는 `[attr]`,`[attr = value]` 형식으로 넣어준다. 예를 들어 `title` 속성이 없는 `<a>`를 선택하고 싶다면 다음과 같이 표현한다.

```css
a:not([title]) {
  /* style rules */
}
```

#### 2.1.5. 동적 가상 클래스 선택자

요소의 상태가 변경될 때마다 적용되는 선택자이다. 이러한 상태는 사용자와의 상호작용을 통해 변경되거나, 현재 문서 상태에 따라 결정될 수 있다.

- `:link`: 방문하지 않은 요소에 스타일을 적용한다.
- `:visited`: 방문한 요소에 스타일을 적용한다.
- `:hover`: 마우스를 올렸을 때 상호작용 중인 요소에 스타일을 적용한다.
- `:active`: 마우스를 눌렀을 때 상호작용 중인 요소에 스타일을 적용한다.
- `:focus`: 키보드 등으로 포커싱이 됐을 때 상호작용 중인 요소에 스타일을 적용한다.
- `:enabled`: 활성화되어 있는 요소에 스타일을 적용한다.
- `:disabled`: 활성화되지 않은(`disabled`속성이 포함된) 요소에 스타일을 적용한다.
- `:checked`: 체크된(`checked`속성이 포함되거나 사용자가 체크한) 요소에 스타일을 적용한다.

### 2.2. 가상 요소 선택자

요소에 내용을 추가하거나 변경하는 데 사용된다. 즉, 실제로 존재하지 않는 요소를 만들어서 스타일을 적용한다.

`::`를 이용해 아래 조건을 붙여서 특정 조건의 요소에 적용한다.

가상 요소는 HTML이 아닌 CSS로 만들어낸, 꾸밈을 위한 요소이기 때문에 브라우저에서 선택(드래그, 복사 등)되지 않는다.

#### 2.2.1. `::before`, `::after`

선언 블록에 `content` 속성을 사용한다.

`content` 속성에 빈 문자열(`""`)을 넣고 고유영역을 `position: absolute`를 통해 없앤 후 구분선, 구분바, 말풍선 테두리 등을 만들 수 있다.

- `::before`: `content` 요소의 시작 부분에 스타일을 적용한다.
- `::after`: `content` 요소의 끝 부분에 스타일을 적용한다.

#### 2.2.2. `::first-letter`, `::first-line`, `::selection`

- `::first-letter`: 첫 번째 글자에만 스타일을 적용한다. 만약 `before`와 같이 쓰인다면 `before`의 `content`속성에 정의된 문자열의 첫 번째 글자에 적용한다.
- `::first-line`: 첫 번째 줄에만 스타일을 적용한다.
- `::selection`: 선택영역(드래그 등)에만 스타일을 적용한다.

## 3. 선택자 결합

### 3.1. 그룹화

`,`를 이용해 둘 이상의 요소를 그룹화할 수 있다.

### 3.2. 하위 선택자 결합

`spacing`을 이용해 첫 번째 요소의 '하위에 존재하는 모든 자식 요소'에 스타일을 적용한다.

### 3.3. 직계 자식 선택자 결합

`>`를 이용해 첫 번째 요소의 '하위에 존재하는 모든 직계 자식 요소"에 스타일을 적용한다.

### 3.4. 형제 선택자 결합

`~`를 이용해 첫 번째 요소의 '뒤쪽 형제 요소'에 스타일을 적용한다.

### 3.3. 인접 형제 선택자 결합

`+`를 이용해 첫 번째 요소의 '뒤쪽에 인접한 상태로 존재하는 한 개의 형제 요소'에 스타일을 적용한다.

## 4. 범용 선택자

`*`를 이용해 모든 요소를 지정한다.

다른 선택자와 결합하여 유용하게 사용할 수 있다.

## 5. 상속을 제어하는 키워드

아래 키워드를 통해 상속을 제어할 수 있다.

해당 키워드들을 `all` 속성과 함께 사용하면 모든 속성을 제어할 수 있다

### 5.1. `initial` 키워드

부모로부터의 상속을 무시하고 초기 상태를 유지한다.

### 5.2. `inherit` 키워드

부모로부터의 상속값을 유지한다.

### 5.3. `unset` 키워드

부모로부터 상속받을 값이 있다면 `inherit`으로 동작하고, 부모로부터 상속받을 값이 없다면 `initial`로 동작한다.
