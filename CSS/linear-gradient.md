# `linear-gradient()`

선형 그라데이션을 생성하는 CSS 함수이다.

`linear-gradient([각도], [색상값1], [색상값2], ...)` 형식으로 작성한다.

다음은 둥근 형태의 인스타그램 로고 예제이다.

```css
.instagram {
  background: linear-gradient(
    174.12deg,
    #3928eb 2.08%,
    #bc1888 20.31%,
    #cc2366 35.42%,
    #dc2743 52.6%,
    #e6683c 76.04%,
    #f09433 96.35%
  );
  border-radius: 50%;
}
```
