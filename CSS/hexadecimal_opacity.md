# rgba()와 16진수를 이용한 투명도 결정

CSS의 요소의 투명도를 결정할 때, `opacity` 속성에 0부터 1까지의 값을 넣어 지정할 수도 있고, `rgba()`의 4번째 인자에 10진수를 넣어서 지정할 수 있다. 또 하나의 방법은 16진수로 표현하는 것이다. `#rrggbbaa` 형식으로 작성하고 'aa'에 아래의 원하는 16진수를 넣어준다.

100% → `FF`  
95% → `F2`  
90% → `E6`  
85% → `D9`  
80% → `CC`  
75% → `BF`  
70% → `B3`  
65% → `A6`  
60% → `99`  
55% → `8C`  
50% → `80`  
45% → `73`  
40% → `66`  
35% → `59`  
30% → `4D`  
25% → `40`  
20% → `33`  
15% → `26`  
10% → `1A`  
5% → `0D`  
0% → `00`
